If you want to run this project you have to do next steps:

1. You have to inport SQL file "./dataElif.sql" to your database
2. You have to change user and password of your databese in "./BackEnd/modules/db.js"
3. For next step you need to open termainal in "./BackEnd" and run next comands 
    "npm i"
    "node server"
4. Now you can open "./FrontEnd/index.html" on some local server (in WebStorm or etc)
